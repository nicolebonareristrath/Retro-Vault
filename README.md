# Retro Media Vault

A multi-page web application and archive dedicated to celebrating the golden age of classic entertainment — featuring timeless films, music, and games from the 20th century.

## Pages
- Home
- Films
- Music
- Games
- Reviews
- Blog
- About
- Contact

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5

## Features
- Live search and category filter
- Contact form with JS validation
- Dark/light mode toggle
- Fully responsive on all devices
